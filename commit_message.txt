[기능 개선] 타이머 고도화 및 운동 데이터/검색 기능 추가

1. 타이머 기능 개선
- 백그라운드 시간 동기화: DateTime 기반 로직으로 변경하여 앱이 잠기거나 내려가도 시간이 정확히 흐르도록 수정
- UI 애니메이션: 게이지가 1초 단위가 아닌 매끄럽게 차오르도록 TweenAnimation 적용

2. 데이터 추가
- 등 운동: 바벨 로우(스미스), 랫 풀다운(패러럴, 맥그립 와이드/클로즈) 추가

3. 검색 편의성 강화
- 초성 검색 시 공백 무시 로직 적용 (예: 'ㄹㅍㄷㅇ' -> '랫 풀다운' 검색 가능)
