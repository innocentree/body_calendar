신규 기능 추가: 최근 운동 우선 표시 및 세트 일괄 변경

1. 최근 운동 우선 표시:
   - 운동 선택 화면에 '최근 운동' 탭을 추가하고 첫 번째 기본 탭으로 설정했습니다.
   - 최근에 수행한 운동 10개를 자동으로 추출하여 보여줍니다.
2. 세트 일괄 변경:
   - 운동 상세 화면의 무게, 횟수, 휴식 시간 설정 영역에 '전체적용' 버튼을 추가했습니다.
   - 클릭 시 현재 설정된 값을 모든 세트에 일괄 적용하며, 휴식 시간의 경우 현재 진행 중인 타이머에도 즉시 반영됩니다.
3. 테스트 코드 수정:
   - MyApp 초기화 관련 위젯 테스트 코드를 현재 앱 구조에 맞게 수정하여 테스트 통과를 보장했습니다.
