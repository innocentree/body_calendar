# 앱 이름 후보

아래 후보들은 현재 일반 웹 검색/앱스토어/깃허브에서 강한 중복이 두드러지지 않는 방향으로 우선 추린 1차안입니다.
최종 상표/출시 전에는 추가 검증이 필요합니다.

## 1) 득근득근
- 톤: 귀엽고 중독성 있음
- 의미: 운동해서 득근하는 리듬감을 이름으로 표현
- 장점: 기억하기 쉬움, 한국어 감성 강함
- 주의: 밈/콘텐츠성 네이밍 느낌이 강함

## 2) 셋로그
- 톤: 직관적, 깔끔함
- 의미: 세트(set) + 로그(log)
- 장점: 앱 기능과 잘 맞음
- 주의: 다소 기능형 이름

## 3) 근캘
- 톤: 짧고 센스 있음
- 의미: 근육 + 캘린더
- 장점: 이 앱의 핵심을 아주 짧게 설명함
- 주의: 발음이 강해서 호불호 가능

## 4) 몸쌓기
- 톤: 담백하지만 재밌음
- 의미: 몸을 쌓아 올린다 + 기록을 쌓는다
- 장점: 브랜딩 확장성 있음
- 주의: 운동 앱처럼 바로 안 읽힐 수 있음

## 5) 오늘몇셋
- 톤: 재밌고 대화체 느낌
- 의미: 오늘 운동 몇 세트 했는지 묻는 말
- 장점: 회화형이라 기억에 남음
- 주의: 다소 가벼운 브랜드 톤

## 6) 근성력
- 톤: 언어유희, 임팩트 있음
- 의미: 근성 + 중력/근력 느낌의 합성
- 장점: 독특함
- 주의: 의미 해석이 약간 갈릴 수 있음

## 클로 추천 Top 3

### A. 근캘
가장 짧고, 앱 정체성이 바로 보입니다.

### B. 셋로그
운동 기록 앱으로 설명력이 좋고, 업데이트/통계/루틴 기능과도 잘 맞습니다.

### C. 득근득근
캐주얼하고 사랑받기 쉬운 이름입니다. 브랜드 캐릭터 만들기 좋습니다.
